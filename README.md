## Setup

### Install

```bash
git clone https://github.com/ironsoul0/bitgaming.me
cd bitgaming.me
yarn
```

### Run

Run local `hardhat` chain for deploying smart contracts.

```bash
yarn chain
```

Deploy smart contracts to local chain.

```bash
yarn deploy
```

Run frontend for interacting with the application.

```bash
yarn dev
```
